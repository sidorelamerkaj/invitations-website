Place your wedding images in this folder with these names for the invite:

  hero.jpg      - Hero section background (landscape, high quality)
  venue.jpg     - Venue section background (full-bleed)
  gallery1.jpg  - Quote section image
  gallery2.jpg  - Gallery large image
  gallery3.jpg  - Gallery image
  gallery4.jpg  - Gallery image
  ceremony.jpg  - Ceremony venue card image
  reception.jpg - Reception venue card image

Any missing image will show a subtle placeholder color. Use JPG or PNG.
